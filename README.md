# Personal Portfolio

A modern and responsive personal portfolio website built using **React** and **Vite**. This project showcases my skills, projects, and experience with a clean UI and optimized performance.

## 🚀 Live Demo

🔗 https://latest-portfolio-ruby-one.vercel.app

## 🛠️ Tech Stack

* React
* Vite
* JavaScript (ES6+)
* HTML5 & CSS3

## ✨ Features

* Responsive design for all devices
* Fast performance with Vite
* Clean and modern UI
* Project showcase section
* Easy to customize and extend

## 📦 Installation & Setup

To run this project locally:

```bash
git clone https://github.com/hello-world123pratik/Latest-Portfolio.git
cd Latest-Portfolio
npm install
npm run dev
```
